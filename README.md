# Auto Mail Campaign Manager

![Logo](public/logo.svg)

Auto Mail Campaign Manager is a tool that is helpful in automating mail sending. Just log in, set your credentials, and send AI-powered mail automatically from your Google account without getting spammed. You can send around 1000 emails in 3.5 days on loop without checking it. Just upload a CSV file, generate or select a mail template, and start the loopâ€”forget about it! Automail will automatically send mails for you.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started
